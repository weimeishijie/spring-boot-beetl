    http://127.0.0.1:8080/index

    共享变量---------> hello beetl
    共享函数--------->
    name = liwenya
    address = 浙江松门
    status = 出差

    constants.getConstant("li") = ONE
    constants.getConstant("wen") = TWO
    constants.getConstant("ya") = THREE
